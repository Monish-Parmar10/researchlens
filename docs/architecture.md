# Research Lens Architecture

## High-Level Architecture

The project consists of three main components:

1. **Frontend (`frontend/`)**
  
   - **Description:** User interface and client-side application.

2. **Backend (`backend/`)**
   - **Description:** Core API, business logic, and database interactions.

3. **AI Service (`ai-service/`)**
   - **Description:** Machine learning models and AI-related functionalities.
